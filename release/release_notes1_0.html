PROTOCOL DESCRIPTIONS

  • See the HBN Data Descriptor for details regarding current HBN protocols and changes over the course of Release 1.0 data collection.

PROTOCOL DISCONTINUATION

  • Staten Island Scanning Site. MRI data are no longer being collected on the Staten Island Site. All scan data from the Staten Island scanner is included in Release 1.0. The Healthy Brain Network will continue scanning at Rutgers University, and plans are underway to open a new scan site at Weill Cornell Medical College in Fall 2017.

DISTRIBUTION NOTE

  • LORIS. Phenotypic data are available for download on LORIS. Due to the sensitivity of the data, users must first download and submit a data usage agreement (available on the phenotypic downloads page) to access. To download the data, follow the instructions on the Scientific Data Portal.
  • Amazon Web Services. Neuroimaging data are stored on Amazon S3 and are available via direct download and file transfer services such as Cyberduck. For more information, visit the neuroimaging downloads page.
