Data Release 2.0¶

RELEASE DATE: 9/14/2017

RELEASE CONTENTS

  • Phenotypic data
    • New datasets: n = 217
    • Total datasets: n = 881
  • Neuroimaging data
    • EEG
      • New datasets: n = 155
      • Total datasets: n = 656
    • MRI
      • New datasets: n = 159
      • Total datasets: n = 767

PROTOCOL ADDITIONS

  • New phenotypic assessments
    • Repetitive Behavior Scale – Revised (RBS-R)
    • Food Frequency Questionnaire (FFQ)
    • Optional follow up language protocol for participants who failed the CELF-5 Screen. Tests include the following:
      • CELF (full assessment)
      • CELF Metalinguistics
      • PPVT-4
      • EVT-2
      • GFTA

PROTOCOL DISCONTINUATION

  • Staten Island Scanning Site. MRI data are no longer being collected on the Staten Island Site. All scan data from the Staten Island scanner is included in Release 1.0. The Healthy Brain Network will continue scanning at Rutgers University, and plans are underway to open a new scan site at Weill Cornell Medical College in Fall 2017.

DISTRIBUTION NOTE

  • COINS. Phenotypic are now available for download on COINS. To download the data, follow the instructions on the Scientific Data Portal. Moving forward, phenotypic data will be uploaded to both COINS and LORIS.
